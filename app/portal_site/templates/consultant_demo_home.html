Consultant sandbox

Three companies. Same reimbursement task. Very different portals.

These local portals are intentionally inconsistent. One is a simple form with a policy pop-up, one is a multi-step workflow, and one is written in Chinese. The future agent will need to read the UI, interpret the policy, and still file the right amount.

Demo setup

  • Both portals live on the same local server.
  • Each portal uses a different flow and different field names.
  • One portal adds a language shift with a Chinese interface.
  • Each portal can submit a reduced claim with an explanation.
